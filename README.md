# Demo Project for Vue 3 & Naive UI & Typescript

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test
```

### Run your end-to-end tests
```
npm run e2e
```

### Lints and fixes files
```
npm run lint
```
